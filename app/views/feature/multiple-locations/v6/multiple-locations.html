Help with cloning an advert

To clone your advert:

  1. Finish creating this advert and submit it to DfE for review
  2. Open the adverts Pending review on your dashboard
  3. Select Manage for this advert
  4. Select Clone advert
  5. Edit the location and other details that need changing, then submit it to DfE
